# 泡浴快递费计算器

一个零依赖的纯静态网页计算器：输入泡浴盒数与收货省份，自动算出 **极兔** 与 **京东** 两家快递的运费，并推荐更便宜的一家。PC 和移动端均自适应。

🔗 在线体验：<https://jsq.nv2.cn>

---

## ✨ 功能

- 输入 **泡浴盒数** + 选择 **收货省份**，一键出价
- 同时计算 **极兔 / 京东** 两家费用，自动 **推荐更便宜** 的渠道并显示差价
- 一盒泡浴按 **0.3 KG** 计算，自动换算计费重量
- 覆盖 **31 个收货省份 / 地区**（始发地固定为江苏省发货）
- 超出快递承运重量时给出提示
- 可展开查看 **完整价格表**
- 移动端优先的响应式设计，PC 自适应加宽
- 纯前端，**无后端、无构建步骤、无第三方依赖**

## 🧮 计费规则

```
单家费用 = 起步价 + MAX(计费重量 − 1KG, 0) × 续重价
计费重量 = 盒数 × 0.3KG          （不向上取整）
```

- 极兔起步价已包含 “+2 元/单” 的加价
- 费用按 1 位小数四舍五入展示

## 🛠 技术栈

纯 HTML + CSS + 原生 JavaScript，单页静态站点。

| 文件 | 作用 |
|---|---|
| `index.html` | 页面结构 |
| `style.css` | 响应式样式（移动端优先） |
| `app.js` | 计算逻辑与交互 |
| `prices.js` | **价格数据（唯一需要维护的文件）** |

## 🚀 本地运行

无需安装任何依赖，任选其一：

```bash
# 方式一：直接用浏览器打开
open index.html

# 方式二：起一个本地静态服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## ✏️ 修改价格

所有费率集中在 [`prices.js`](./prices.js)。修改对应省份的 `jt`（极兔）/ `jd`（京东）数值即可，无需改动其它代码：

```js
{ dest: "上海", jt: { qi: 6, xu: 0.9, max: 30 }, jd: { qi: 7.2, xu: 1, max: 30 } }
//                    ↑起步价  ↑续重价/KG  ↑最大可寄重量(KG)
```

改完刷新页面即生效。
